BrIW

An application to manage tea rounds.

BrIW is a command-line application which allows you to create a tea round for a group of people.


When running the application, you'll be greeted with a set of menus which let you edit the people in your group, their drink preferences, and the list of drinks available to make.

You can find more instructrions for using the application here.
